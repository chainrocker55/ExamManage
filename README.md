# Quick start

Step 1
> git clone https://github.com/nine9517/ExamManage.git

Step 2

> cd ExamManage

Step 3

> npm i

Step 4

> cd public

Step 5

> npm i

Step 6

> cd .. && npm start

## Testing

Step 1

> npm i -g mocha

Step 2

> npm test

## Configuration

You can change the settings on the file app.json in folder configuration
